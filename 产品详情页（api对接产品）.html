说明：


1.已对接的产品展示新产品详情页（如左图所示，未对接API的产品仍沿用旧产品详情页），用户进入详情页时获取用户的定位权限；


2.新产品详情页中可贷额度、贷款期限由商务收集后后台进行配置，均支持一个范围；产品详情模块展示后台申请条件中填写的内容，有商务收集填写，产品详情模版默认为收起状态，点击后展开；


3.若合作机构返回的产品参数类型为数组，则用户可通过点击金额/期限，弹出弹窗，用户可上下滑动进行金额/期限的选择；若合作机构返回的产品参数为范围值，则在范围内用户可自由编辑输入数值；


4.进入详情页后调用获取协议接口，展示在详情页最下方。


5.小白来花提供加密后的手机号调用合作机构的可申请用户接口，判断该用户是否可申请该产品，若可申请则更新产品信息，若不可申请则返回不可申请原因。


6.不可申请用户根据返回的不可申请原因弹出相应的提醒弹窗，点击“我知道了”，返回至上一页面；

7.点击各项进行认证，其中实名认证7天过期（过期显示“已过期”状态），运营商报告30天过期,同时在此页面调用合作机构的银行卡列表接口，若查的合作机构借款卡列表中存在一张默认卡且该卡通过我们平台绑定或校验，则显示已认证；否则，显示未认证。


8.全部认证成功后，按钮变成“立即申请”，点击时调用合作机构借款试算接口，获取到账金额、利息和服务费、还款金额以及还款日期，弹出订单确认弹窗；


9.点击“确认”，调用合作机构的订单基本信息推送接口（实名认证信息、基本信息、联系人信息、银行卡信息、通讯录）和补充信息推送接口（运营商报告），合作机构响应是否推单成功，成功则进入申请成功页，失败则返回失败原因。


10.若合作机构未正常接收到订单补充信息，可主动调用小排钱包的订单补充信息获取接口，主动获取补充信息（运营商报告）。
